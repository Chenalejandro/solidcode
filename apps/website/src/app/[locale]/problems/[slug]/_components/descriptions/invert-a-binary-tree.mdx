## Invertir árbol binario

Dado un árbol binario, retornar el árbol invertido.

### Ejemplos

Entrada:


```
   1
  / \
 2   3
```

Salida:
```
   1
  / \
 3   2
```

Entrada:

```
   1
  / \
 2   3
  \
   4
  / \
 5   6
  \
   7
```

Salida:

```
   1
  / \
 3   2
    /
   4
  / \
 6   5
    /
   7
```
