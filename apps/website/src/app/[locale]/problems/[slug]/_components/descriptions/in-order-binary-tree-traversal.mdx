## Recorrido del 치rbol binario en orden

Dado un 치rbol binario, retornar un arreglo con todos los valores de los nodos del 치rbol, recorriendo el 치rbol en orden.

### Ejemplos

Entrada:

```
   1
  / \
 2   3
```

Salida: `[2, 1, 3]`

Entrada:

```
   1
  / \
 2   3
  \
   4
  / \
 5   6
  \
   7
```

Salida: `[2, 5, 7, 4, 6, 1, 3]`
