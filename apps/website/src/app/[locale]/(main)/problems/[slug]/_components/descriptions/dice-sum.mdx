## Suma de los dados

Dado n dados de k caras, calcular todas las maneras posibles de conseguir la suma total s con una sola tirada.
Consideramos que los dados son distinguibles.

### Ejemplos

Entrada:

`cantDados = 2`

`cantCaras = 2`

`sumaRequerida = 3`

Salida: `2`

Explicación: Hay dos combinaciones posibles para que dos dados sumen 3. Uno es que el dado1 dé 1 y el dado2 dé 2. Otro es que el dado1 dé 2 y el dado2 dé 1.
